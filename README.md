
# AppAngularEPrimeNg

Parte front-end do sistema para cadastro de Produto, Fornecedor e Nota fiscal

## Executando o aplicativo no modo dev

```bash
ng serve
```

Após subir a aplicação entre nessa URL `http://localhost:4200/`.

## Oque foi usado

- Angular 19.1
- PrimeNg 19
